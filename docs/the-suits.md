Cada traje se corresponde con elementos naturales.


También se pueden utilizar para corresponder a los signos del zodíaco.


Utilice lo siguiente para comprender mejor las tarjetas




Varitas:
     Elemento - Fuego
     Signos: Aries, Leo, Sagitario

Las varitas están relacionadas con el crecimiento, la creatividad y la pasión.

Tazas:

     Elemento - Agua
     Signos: Piscis, Cáncer, Escorpio

Las copas están relacionadas con las emociones, la conexión, la intuición, las artes, los sueños.

Espadas:

     Elemento - Aire
     Signos: Acuario, Géminis, Libra

Las espadas representan la mente, el pensamiento racional, la lucha, el conflicto.

Pentáculos:

     Elemento - Tierra
     Signos: Tauro, Virgo, Capricornio

Pentáculos (o Discos) representan posesiones materiales, dinero, trabajo, trabajo, hogar.
